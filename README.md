# TFG-Comercio-Internacional-Redes-Complejas
Código y datos del Trabajo de Fin de Grado: "Análisis del comercio internacional basado en herramientas matemáticas de redes complejas" del Grado en Matemáticas.

## Contenido

- Construcción de la red a partir de los datos de UN Comtrade
- Limpieza y depuración de los datos
- Reducción de la dimensionalidad de la red
- Cálculo de centralidades
- Detección de comunidades mediante Louvain
- Generación de figuras
